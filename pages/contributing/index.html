Contributing

PowSyBl commitments

Code of conduct

This project applies the LF Energy Code of Conduct. By participating, you are expected to uphold this code. Please report unacceptable behavior to the project’s Technical Steering Committee.

License

PowSyBl is an open source framework licensed under the Mozilla Public License 2.0. By contributing to PowSyBl, you agree to the terms and conditions for your present and future contributions submitted to PowSyBl.

Technical charter

By contributing to PowSyBl, you are expected to uphold the technical charter.

Developer Certificate of Origin

Contributors must sign off their commits to assert that they are the creator of the contribution, and that they allow PowSyBl to use their work.

Continuous Integration

Continuous Integration helps us maintain the quality of the code through automatic checks.

How to contribute

Whether you want to report a bug or suggest an enhancement, follow the two steps described below.

1. Contact the team

This project applies the LF Energy Code of Conduct. By participating, you are expected to uphold this code. Please report unacceptable behavior to the project’s Technical Steering Committee.

2. Open issues / Pull Requests

Once you have contacted the team, you may open issues and even contribute to the code! Check our contribution guidelines.
